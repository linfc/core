For each supported platform:

1. Download Eclipse classic for the specific platform

2. Uncompress the file

3. Rename eclipse to classic

4. Add the link files for: 

     asm, byteviz, egit, elt, gef, mylyn, subversive, texlipse
     
     Each link file contains the following: path=../eclipse-plugins-essential/<name>
       where <name> is either asm, byteviz, egit, elt, gef, mylyn, subversive, or
       texlipse, respectively

5. In eclipse.ini, change the following arguments:
   * --launcher.XXMaxPermSize 512m
   * -Xms128m
   * -Xmx1024m

6. For Mac versions:
   * Delete the classic/eclipse symlink
   * Copy sireum-distro/scripts/eclipse/mac/eclipse to classic/
   * Copy sireum-distro/scripts/eclipse/mac/eclipse.sh to classic/Eclipse.app/Contents/MacOS/
   * Edit the classic/Eclipse.app/Contents/Info.plist so the executable is eclipse.sh
   
   For Linux versions:
   * Copy sireum-distro/scripts/eclipse/linux/eclipse.sh to classic/
   * Copy sireum-distro/scripts/eclipse/linux/README.TXT to classic/
   
   For Windows versions: 
   * Copy sireum-distro/scripts/eclipse/windows/eclipse.bat to classic/
   * Copy sireum-distro/scripts/eclipse/windows/README.TXT to classic/

7. Zip the classic directory

   * For Mac 64-bit    : zip -r classic-mac64.sapp classic  
   * For Mac 32-bit    : zip -r classic-mac32.sapp classic  
   * For Linux 64-bit  : zip -r classic-linux64.sapp classic  
   * For Linux 32-bit  : zip -r classic-linux32.sapp classic  
   * For Windows 64-bit: zip -r classic-win64.sapp classic  
   * For Windows 32-bit: zip -r classic-win32.sapp classic  

8. Transfer all the .sapp files to ~santos/stash/apps.sireum.org/Eclipse/
   in the CIS Linux system
